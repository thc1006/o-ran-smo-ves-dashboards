# Code of Conduct

This project adopts the
[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
unchanged and in full.

The canonical text lives at the URL above; it is not copied here to
avoid drift between our copy and the upstream revision.

## Enforcement contact

Report abuse or conduct issues to the project maintainer at
`hctsai1006@cs.nctu.edu.tw`. Reports are reviewed in confidence and
responded to within seven days.

## Scope

Applies to every space managed by this project: the GitHub repository,
issue tracker, pull request reviews, release notes, and any off-platform
interactions concerning the project.
